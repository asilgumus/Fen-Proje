# Fen Proje

This repository contains a simple web project consisting of HTML, CSS, and JavaScript files. The project includes the following files:

- `index.html` - The main entry point of the website.
- `start.html` - An additional HTML page possibly serving as a start or introductory page.
- `style.css` - The cascading style sheet to style the HTML content.
- `script.js` - JavaScript file to add interactivity and dynamic functionality.
